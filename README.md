# grounds

个人学习仓库 —— 用对话学知识，用 git 管笔记。

**入口**：[`AGENTS.md`](AGENTS.md) — Claude Code 和 Codex 通用，包含完整约定与七个 skill。

## 结构

```
wiki/    学习笔记（按主题分目录）
paper/   论文笔记（按主题分目录）
video/   视频笔记成品（每个视频一个目录）
raw/     原始资料（只增不删；按来源分 topic/papers/videos）
```

## 七个 Skill

- 自动触发：`learn` · `capture` · `lint` · `query`
- 手动 / `$` 触发：`paper-learn` · `bilibili-render-pdf` · `youtube-render-pdf`

详见 [AGENTS.md](AGENTS.md)。
